# 🌤️ Design Patterns – Observer Pattern  
## Assignment: Weather Station System

---

## 📘 Scenario

A weather station continuously measures weather data.  
Different display systems and services must be updated automatically when weather conditions change.

The system has a **WeatherStation** that tracks:

- Temperature (double)
- Humidity (double)
- Pressure (double)

When the weather data changes, the following systems must be notified:

- **CurrentConditionsDisplay**
- **StatisticsDisplay**
- **ForecastDisplay**

---

## 🎯 Task

Create a **UML class diagram** that models this system using the **Observer Pattern**.

Your diagram must include:

- An **Observable** interface
- An **Observer** interface
- The WeatherStation class
- The three display systems

---

## 📐 Requirements

- Use correct UML notation
- Show all relevant attributes and methods
- Show correct relationships (realization, aggregation)
- Apply the Observer Pattern correctly
