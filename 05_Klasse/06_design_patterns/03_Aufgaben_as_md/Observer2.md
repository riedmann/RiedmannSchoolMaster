# 🏠 Design Patterns – Observer Pattern  
## Assignment: Home Automation System

---

## 📘 Scenario

A modern home automation system continuously monitors the temperature in the building.  
Different systems must react automatically when the temperature changes.

The system has a **TemperatureSensor** that tracks:

- Current temperature (double)

When the temperature is **greater than 20 degrees**, the following systems must be notified:

- **Blinds**
- **Lights**
- **Doors**

---

## 🎯 Task

Create a **UML class diagram** that models this system using the **Observer Pattern**.

Your diagram must include:

- An **Observable** interface
- An **Observer** interface
- The TemperatureSensor class
- The three observer systems (Blinds, Lights, Doors)

---

## 📐 Requirements

- Use correct UML notation
- Show all relevant attributes and methods
- Show correct relationships (realization, aggregation)
- Apply the Observer Pattern correctly
